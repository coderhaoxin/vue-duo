### vue-duo

* You can handle complex data flow with awesome `rxjs`.
  - [vuejs/vue-rx](https://github.com/vuejs/vue-rx) is a RxJS binding for Vue.js.
* `vue-duo` provide a tiny store for simple data flow, works fine with `vue-rx`.
